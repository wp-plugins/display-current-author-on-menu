Tiny plugin that allows you to display current author's name(Display Name) on menu.


== Description ==
This simple plugin allows users to display current author name on menu after logged in. Ideal for  Multi Author Sites. 

Just add [display-author] shortcode to your menu's menu item level and you are good to go.

== Installation ==
This section describes how to install the plugin and get it working.


 Search and install from your admin dashboard
 
 or

1. Unzip and Upload the display-current-author-on-menu folder to your plugin directory.

2. Activate it. Paste [display-author] short-code on which menu you want.

== Frequently Asked Questions ==
What should i do after activating the plugin?

Add [display-author] shortcode to your menu's menu item level where you want to display the name.
